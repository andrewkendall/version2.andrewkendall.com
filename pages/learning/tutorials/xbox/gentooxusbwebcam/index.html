Setting up a USB webcam under Gentoox

This tutorial was written and successfully tested using a Logitech QuickCam Pro it should also support most other Logitech cameras.

In order to use other brands of webcam you may need to install other drivers, if so consult http://www.exploits.org/v4l/, find the name of your cameras driver and then substitute step 3 for the relevant driver install. I’m afraid I can’t help with other cameras but I would appreciate your feedback on what worked and what didn’t.


For this you will need

1. Xbox running Gentoox (http://gentoox.shallax.com)
2. USB webcam
3. Xbox female USB adapter (http://www.lik-sang.com)


Steps

1. Install the webcam software:

# emerge camsource


2. Copy the camsource config file:

# cp /etc/camsource.conf.example /etc/camsource.conf


3. Install the webcam driver.

# emerge qce-ga

4. Now plugin the webcam and reboot the Xbox. To start camsource manually just type:

# camsource

Then open your web browser and go to the following:

http://yourxboxip:9192/

There are other URLs such as http://yourxboxip:9192/large and http://yourxboxip:9192/largequal/ for more info see http://camsource.sourceforge.net/


That's it, hope it helps :-)
