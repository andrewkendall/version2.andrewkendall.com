Objective

Setup a CGI script to create your own weblog.

For this you will need

A Machine running Apache, FTP Server and Perl.
FTP Client (www.cuteftp.com)
Greymatter CGI script (Download)

Further resources

Free CGI scripts (www.cgi-resources.com)

Step by step guide

  1. Downloading Greymatter
  2. Unzipping And Changing Path To Perl
  3. Starting FTP And Apache
  4. Uploading Files
  5. Setting Up Greymatter

Once Greymatter is setup you may modify the html template as much as you like.

If all this is successful try some more CGI scripts from www.cgi-resources.com
